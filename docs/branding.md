# Engage Solar — Branding

## Conceito

**Engage** — plataforma de relacionamento, automação WhatsApp e recuperação de vendas para energia solar.

## Sensação visual

- Energia
- Tecnologia
- Crescimento
- Confiabilidade
- Dados

## Paleta

| Token | Hex | Uso |
|-------|-----|-----|
| `--es-primary-deep` | `#12355B` | Sidebar, títulos |
| `--es-primary` | `#1E5AA8` | Links, barras, destaques |
| `--es-solar` | `#F7B500` | CTA, item ativo, acentos |
| `--es-energy` | `#FF8C00` | Gradientes, alertas quentes |
| `--es-bg` | `#F7F9FC` | Fundo da área principal |
| `--es-surface` | `#FFFFFF` | Cards |
| `--es-text` | `#1F2937` | Texto principal |
| `--es-text-muted` | `#6B7280` | Legendas |

## Header

```css
background: linear-gradient(90deg, #12355B, #1E5AA8, #F7B500);
```

## Tipografia

System UI stack (Inter-like): `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`

## Logo

Sol estilizado + wordmark **Engage** / **SOLAR** (secundário em amarelo).
